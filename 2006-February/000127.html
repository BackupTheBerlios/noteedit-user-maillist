I use Debian unstable, so I downloaded the source package, I patched the wrong 
line by hand, then I rebuilt the package :

apt-get source noteedit
cd noteedit-2.8.0
fakeroot dpkg-buildpackage
cd ..
su -c dpkg -i noteedit*.deb
