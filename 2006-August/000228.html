Sorry I didn't finish my earlier reply, Chuckk;  my keyboard batteries
went out suddenly. Any way, try using Ctrl while mouse clicking the
element you want to delete.  For beams, go to edit mode, highlight the
first notein the beam series, and turn off the beam button on the tool
bar.  To delete slurs, if edit mode does not work, delete the first note
of the slur, then replace it.
