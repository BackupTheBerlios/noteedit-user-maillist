Try pressing "Ctrl" key while clicking on what you want to delete.  This
worked for me when I was using noteedit 2.8.0 on SuSE 10.1. Since I
upgraded to 2.8.1 the delete key works for most things. There seems to
be a disagreement between some versions of noteedit and KDE. You have to
be using the proper versions of both to get the best results.
