Hm... wierd. I forwarded the problem to the LilyPond-devel mailing list. 
They'd probably know better than me:).

About the \drummode mode, I think that NoteEdit really needs the concept 
of contexts like in LilyPond in the future. So if you'd insert a 
percussion staff, all the notes will be drum notes, if you insert an 
ordanary staff, you'd get normal notes and 5-line staff, if you insert 
tablature, you'd get 6-line staff and tablature notation etc. Until 
then, using different style of noteheads is the best possible solution IMO.


Regards.
