noteedit-user-admin at berlios.de wrote:

| What puzzled me: A \PianoStaff only allows positioning of lyrics
| below the pair of staff. Maybe this is also a candidate for a
| question on the lilypond list. And bracing and bracketing when some
|  staffs are not selected I will work out soon (score layout). And
| which lilypond versions backwards we could not test/brake?

Hm... why should you have lyrics in piano staff anyway? All choir and
other sung music is always inside a bracket (and not brace - brace is
stronger. It always includes single instrument - piano, organ etc.).
But I don't know if that's what Lilypond wanted to really implement in
this case. On the other hand, I don't remember an example where you
should use \pianoStaff and needed lyrics for it. \choirStaff should
support lyrics between the pair of staves instead.

About Lily versions, under Debian the problem is that Debian testing
and unstable both include 2.2.6 version of Lilypond. So we cannot make
any changes until both testing and unstable include 2.4.x in their
repository if we want to stick to and only to 2.4.x. However, we can
drop support for versions prior to 2.2.x. What about Fedora, SuSE,
Mandrake?
