Yes, that's the structure I had in mind as well. Everything is moving
around contexts: staff context, lyrics context, chords context,
tablature context, general bas markings context, harmonic analysis
context etc. Lyrics with multiple stanzas are treated as a separate
Lyrics contexts for each stanza. That's what you actually implemented
and it's perfectly ok.

One question though: Does numbers 1. and 2. (stanza numbers) appear at
the beginning of the stanza (or after it?)? Because I think you should
write the \set stanza = "1." BEFORE the actual lyrics... but I'm not
sure:).

noteedit-user-admin at berlios.de wrote:

> Hi all,
>
> before completing the implementation I would like to share with you
a lilypond export format, mainly a modfied \score block, which will
give us more features, I think. It allows us to put lyrics in print
not only beneath the staff, but also over or in mixed fashion anywhere
in the score. I attach an exported .ly file where you can see the
basic format and maybe do experiments.
>
> To have in noteedit lyrics always below the music is perfectly okay,
because in noteedit you are an aranger. But in printout it should be
easily possible to make everything more compact, for example, combine
two voices and flip the lyrics as you whish. I just inted to export
this format also as a helpful hint for the user that he might be able
to modify the exported file and create a patch. It took me quite some
time to extract everything out of the lilypond documention. And
Matevž, this is the format I once mentioned.
>
> Any thoughts about it are welcome. I hope to close bug #3403 with
this whithin next week. I just would like to mention, that multi voice
(not chords) usage crashed quite easily for me, so I avoid it currently.
>
> Best Regards,
> Georg

>%                                  %
>%   LilyPond output generated by   %
>%           "noteedit"             %

>
>\version "2.4.2"
>
>#(set-global-staff-size 16)
>
>flat = \markup { \raise #0.4 \smaller \musicglyph #"accidentals--2" }
>sharp = \markup { \raise #0.6 \smaller \musicglyph #"accidentals-2" }
>textcodaysym = \markup { \hspace #1 \raise #1.1 \musicglyph
#"scripts-coda"}
>
>StaffA = \relative c' {
>	\clef G
>	\key g \major
>	\time 4/4 
>	                   
>	g'4 b8 b d4 e | f! e d b |   % 3
>	g1 | g 
>	\bar "|."
>}
>StaffATextA = \lyricmode {
>		Dum dum dum dum dum dum dum dum 
>		dum dum doo. 
>		}
>StaffATextB = \lyricmode {
>		Tsk da da da tsk da da da 
>		dai daa soo
>		}
>StaffB = \relative c' {
>	\clef G
>	\key g \major
>	\time 4/4 
>	                   
>	g'4 b d e | f! e8 e d4 b |   % 3
>	g1 | g 
>	\bar "|."
>}
>StaffBTextA = \lyricmode {
>		shum dum shum dum shum dum dum dum 
>		dum dum soo. 
>		}
>StaffBTextB = \lyricmode {
>		Tsk da dum dum tsk da dum dum 
>		dai daa doo
>}
>\score {
>	\relative <<
>		\context Staff = cStaffAA <<
>			\context Voice = cStaffAA \StaffA
>		>>
>		\context Lyrics = cStaffAA { \set stanza = "1." }
>		\context Lyrics = cStaffAB { \set stanza = "2." }
>
>		\context Staff = cStaffBB <<
>			\context Voice = cStaffBB \StaffB
>		>>
>		\context Lyrics = cStaffBA { \set stanza = "1." }
>		\context Lyrics = cStaffBB { \set stanza = "2." }
>
>		\context Lyrics = cStaffAA \lyricsto cStaffAA \StaffATextA
>		\context Lyrics = cStaffAB \lyricsto cStaffAA \StaffATextB
>		\context Lyrics = cStaffBA \lyricsto cStaffBB \StaffBTextA
>		\context Lyrics = cStaffBB \lyricsto cStaffBB \StaffBTextB
>	>>
>}
>%\score {
>%	\simultaneous {
>%		\set Score.skipBars = ##t
>%		\set Score.melismaBusyProperties = #'()
>%		\oldaddlyrics 
>%		\context Staff="StaffA" \StaffA
>%		\context Lyrics="StaffAText" \StaffAText
>%		\oldaddlyrics 
>%		\context Staff="StaffB" \StaffB
>%		\context Lyrics="StaffBText" \StaffBText
>%	}
>%	\layout {
>%		linewidth = 170.000 \mm
>%		textheight = 250.000 \mm
>%	}
