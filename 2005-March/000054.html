I checked the Lily mailing list later and found out the same thing you
did. The style of noteheads is hardcoded to the voice. So if you
change the style of noteheads, it changes the style of the whole voice
automatically, so there's nothing useful here, we could make. I think
the best suitable option is not to have different style of noteheads
in one voice at all. The solution would be for eg. adding additional
voice only for that part of the score (and of course aligning the
stems, so it'd look like one voice). But you should do all that
manually for now.
