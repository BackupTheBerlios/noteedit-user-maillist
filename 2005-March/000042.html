Hi all,

before completing the implementation I would like to share with you a 
lilypond export format, mainly a modfied \score block, which will give 
us more features, I think. It allows us to put lyrics in print not only 
beneath the staff, but also over or in mixed fashion anywhere in the 
score. I attach an exported .ly file where you can see the basic format 
and maybe do experiments.

To have in noteedit lyrics always below the music is perfectly okay, 
because in noteedit you are an aranger. But in printout it should be 
easily possible to make everything more compact, for example, combine 
two voices and flip the lyrics as you whish. I just inted to export this 
format also as a helpful hint for the user that he might be able to 
modify the exported file and create a patch. It took me quite some time 
to extract everything out of the lilypond documention. And Matevž, this 
is the format I once mentioned.

Any thoughts about it are welcome. I hope to close bug #3403 with this 
whithin next week. I just would like to mention, that multi voice (not 
chords) usage crashed quite easily for me, so I avoid it currently.

Best Regards,
Georg
